Essa pasta contém os relatórios de optimização do Lighthouse referentes ao projeto ecommerce-project, para o curso de Front-End da EBAC.

Melhorias realizadas:
- Limpeza de imports não utilizados
- Remoção de comentários
- Limpeza de CSS
- Conversão de todas imagens dos produtos para .webp
- Minificação do código (realizada automaticamente pelo React)
